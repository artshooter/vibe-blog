# <img src="./app/icon.png" width="32" height="32" alt="icon" /> Vibe Blog

**氛围博客** - 每篇文章都有独特的视觉风格和交互体验，拒绝千篇一律的模板。

- 视觉风格匹配文章情绪
- 交互组件帮助理解内容
- 形式服务于内容
- RAG 智能问答（基于文章内容）

## 快速开始

1. 切换到 `blank` 分支（干净的起点）
2. 打开 Claude Code，告诉 AI：「写文章」
3. 跟着 AI 流程走就行

![Quick Start](./quick-start.png)

## 本地运行

```bash
pnpm install
pnpm dev
```

访问 http://localhost:3000

## 技术栈

Next.js 16 · React 19 · TypeScript · Tailwind CSS · Framer Motion · next-intl · LanceDB
