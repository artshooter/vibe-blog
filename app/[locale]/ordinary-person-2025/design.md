# 设计文档：普通人的 2025

## 1. 内容理解

### 主题
裸辞后的迷茫与自我发现。作者从不断尝试又放弃的"猴子掰玉米"状态，到发现自己真正害怕的是努力、是真实的反馈，最终接受"我是个普通人"。

### 情绪曲线
```
迷茫焦虑 → 反复尝试 → 自我怀疑 → 顿悟剖析 → 释然接受
   6月        6-11月        9月婚礼后      12月         现在
```

### 目标读者
- 正在经历职业迷茫的年轻人
- 有过类似"聪明但没努力"心理的人
- 对自我认知感兴趣的读者

---

## 2. 设计直觉

### 第一感受
读完文章，脑海中浮现的是：一个人坐在书桌前，窗外是冬天的阳光，手边是写了又划掉的笔记本。安静、真诚、有点脆弱但很勇敢。

### 脑海中的画面
- 米白色的纸张，有些泛黄
- 手写的字迹，有圈有划
- 冬日下午柔和的光线
- 散落的便签和涂鸦

### 色彩感受
- 主色：温暖的米白/象牙白 - 纸张的感觉
- 辅助：暖灰色 - 铅笔的质感
- 点缀：淡淡的蓝或橙 - 用于强调

---

## 3. 设计方案

### 3.1 视觉风格

**整体风格**：年终手记 / 私人日志

**色彩方案**：
- 背景：`#FDFBF7`（象牙白）
- 正文：`#3D3D3D`（深灰，非纯黑，更柔和）
- 强调：`#E07A5F`（暖橙红，用于重点句）
- 辅助：`#81B29A`（淡绿，用于时间节点）
- 引用背景：`#F4F1EA`（浅米色）

**字体**：
- 标题：思源宋体 / Noto Serif SC - 有文化感但不老气
- 正文：系统默认字体 - 保证可读性
- 强调文字：可考虑手写风格字体（谨慎使用）

**视觉元素**：
- 纸张纹理作为背景（subtle）
- 重点段落用手绘下划线效果
- 时间节点用简单的圆点 + 日期

**风格关键词**：真诚、安静、内省、温暖、不完美

### 3.2 交互设计

**核心交互点**：

1. **项目墓地 / 时间线**
   - 展示从6月到12月尝试的各个项目
   - 可以是一个水平滚动的时间线，或者垂直的"墓碑"列表
   - 每个项目显示：名称、简述、结局标签（"放弃"、"搁置"等）
   - 交互：hover 显示更多细节

2. **"聪明但没努力"心理分析**
   - 那段核心顿悟可以做成一个展开的分析框
   - 点击展开查看 AI 解析部分
   - 或者用视觉上的"层层剥开"效果

3. **两层生活模型**
   - 简单的双层可视化
   - "支撑层"在下，"探索层"在上
   - 可以是静态插图，带简单的 hover 效果

**原则**：交互要克制，这篇文章的力量在于文字本身，交互只是辅助理解，不抢戏。

### 3.3 沉浸式设计

**核心隐喻**：年终手记

将整篇文章呈现为一份私人年终手记：
- 页面像是翻开一本笔记本
- 有"年份"的感觉：2025
- 重点句像是被作者用笔圈出来的
- 结尾有一种"合上本子"的完结感

**导航逻辑**：
- 简单的垂直滚动
- 时间节点作为锚点（6月、9月、12月）
- 不需要复杂的导航，让读者沉浸在阅读中

**滚动体验**：
- 平滑滚动
- 重点段落进入视口时有轻微的淡入效果
- 不要过度动画，保持安静的阅读体验

### 3.4 响应式设计

**桌面端**：
- 最大宽度 720px，居中显示
- 两侧留白，像打开一本书
- 时间线可以水平展示

**移动端**：
- 全宽显示，适当边距
- 时间线改为垂直堆叠
- 字号适当增大，保证可读性
- 触摸交互：轻触展开详情

## 4. 技术实现

### 核心组件
```
app/components/ordinary-person-2025/
├── index.tsx          # 元数据导出
├── Hero.tsx           # 首页卡片
├── Content.tsx        # 文章主体
├── Timeline.tsx       # 项目时间线组件
└── TwoLayers.tsx      # 两层生活可视化
```

### 依赖库
- `framer-motion`：轻量动画（淡入效果）
- `next-intl`：国际化
- 无需额外 UI 库，用 Tailwind 实现

### 性能考虑
- 背景纹理使用 CSS 渐变模拟，避免图片
- 动画使用 `will-change` 和 `transform`
- 组件按需加载
