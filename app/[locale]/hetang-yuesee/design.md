# 荷塘月色 - 设计文档

## 内容理解

朱自清的经典散文，记录了一次月夜独自漫步荷塘的心灵旅程。

**核心情绪**：
- 开篇"心里颇不宁静"——内心的躁动与不安
- 漫步中逐渐获得片刻宁静
- 歌声唤起回忆与感慨
- 最终回归平静的日常

**关键意象**：
- 满月、月光（银白、清冷、朦胧）
- 荷塘、荷叶（静谧、层叠）
- 杨柳、树影（如烟雾般）
- 蛙声、蝉声（热闹却不烦躁）
- 远处的歌声（清远、飘渺）

## 设计直觉

### 第一感受

看到这篇文章，脑海中浮现的是：

**画面**：深蓝色的夜幕下，银色月光洒在层层叠叠的荷叶上，远处杨柳的剪影如烟如雾，一条小路蜿蜒向前。

**颜色**：
- 深邃的靛蓝色夜空 `#0a1628`
- 银白的月光 `#e8f4f8`
- 墨绿的荷叶 `#1a3a2a`
- 朦胧的灰紫 `#4a4a6a`

**感觉**：宁静、朦胧、略带忧郁、独处的慰藉

**节奏**：缓慢、沉浸、像夜间散步一样不急不躁

## 视觉风格

### 色彩方案

```
主色调：深夜蓝 #0a1628 → #1a2a48（渐变夜空）
点缀色：月光银 #e8f4f8（文字、光晕）
辅助色：荷叶墨绿 #1a3a2a（装饰元素）
氛围色：烟雾紫灰 #4a4a6a（过渡、阴影）
```

### 字体选择

- **标题**：思源宋体 / Noto Serif SC（古典韵味）
- **正文**：思源宋体，1.8倍行高（舒缓的阅读节奏）
- **特殊段落**：可用楷体增添手写质感

### 视觉元素

- **月亮**：画面上方的圆月，随滚动微微升高
- **荷叶剪影**：底部或侧边的装饰性荷叶轮廓
- **光晕效果**：月光的柔和光晕，营造朦胧感
- **波光粼粼**：荷塘水面的微光效果

## 交互设计

### 1. 月亮升落

随着阅读进度，月亮位置从低到高再逐渐变淡：
- 文章开头：月亮刚升起，位置较低
- 文章中段：月亮高悬
- 文章结尾：月光变淡，星星出现

### 2. 声景交互

在特定段落触发环境音效（可选开关）：
- 蛙声与蝉声的层叠
- 远处飘来的歌声

### 3. 光影变化

页面背景的月光区域随滚动变化：
- 营造"月光在移动"的感觉
- 使用 CSS 渐变或 canvas 实现

### 4. 段落淡入

文字如月光般缓缓浮现：
- 滚动到视口时平滑淡入
- 配合轻微的上浮动画

## 沉浸式设计

### 核心隐喻：月夜漫步

将阅读体验设计成一次**跟随作者的散步**：

**Hero 区域**（进入荷塘）：
- 深邃的夜空背景
- 一轮明月悬挂
- 标题"荷塘月色"以月光色呈现
- 远处荷塘的朦胧剪影

**正文区域**（漫步荷塘）：
- 保持夜色氛围的深色背景
- 文字如月光般明亮但不刺眼
- 侧边可有隐约的荷叶、杨柳装饰
- 滚动时月光光斑微微移动

**结尾区域**（回到家门）：
- 氛围逐渐收束
- 暖色调的灯光微微出现
- 回归宁静的日常

### 整体氛围

- **不用纯黑背景**：用深蓝紫色，保持夜晚的温度
- **文字发光效果**：轻微的 text-shadow 模拟月光照射
- **朦胧感**：适当使用 blur 和半透明层

## 响应式考虑

### 桌面端

- 宽阔的视野，月亮和荷塘可以有更多空间展开
- 两侧可有淡淡的荷叶装饰
- 文字宽度控制在 65-75 字符

### 移动端

- 简化装饰元素，保留核心的月亮和光影
- 文字占据更多屏幕宽度
- 动画效果适当精简以保证性能
- 触摸滚动保持流畅

## 技术实现

### 组件结构

```
app/components/hetang-yuesee/
├── Hero.tsx           # 封面：月夜荷塘场景
├── ContentSection.tsx # 正文容器：深色背景+月光效果
├── Moon.tsx           # 月亮组件：位置随滚动变化
├── LotusLeaves.tsx    # 荷叶装饰：SVG剪影
└── MoonlightEffect.tsx # 月光效果：光斑、光晕
```

### 关键技术

1. **月亮升降**：使用 Framer Motion 的 `useScroll` 和 `useTransform`
2. **光影效果**：CSS radial-gradient + 动态位置
3. **文字淡入**：Intersection Observer + CSS transition
4. **荷叶剪影**：SVG 图形，可复用不同形态

### 性能优化

- 使用 CSS 变量控制颜色，方便主题调整
- 动画使用 transform 和 opacity，避免重排
- 移动端减少粒子/光斑数量
- 图片资源使用 WebP 格式

## Hero 组件规格

### inHome={true} 模式（首页卡片）

- 高度固定 `h-[400px]`
- 展示：月夜荷塘的简化场景
- 月亮、几片荷叶剪影、标题
- 整体色调统一，点击可跳转

### inHome={false} 模式（文章详情页）

- 全屏高度 `min-h-screen`
- 完整的月夜场景
- 月亮可动态变化
- 可能包含简短的引言
