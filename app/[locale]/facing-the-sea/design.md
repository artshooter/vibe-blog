# 面朝大海，春暖花开 - 设计文档

## 内容理解

海子写于1989年1月13日的诗歌，两个月后诗人离世。这首诗表面洋溢着对幸福生活的向往，实则蕴含着深刻的孤独与疏离。

**核心张力**：
- "从明天起"——为什么是明天，而不是今天？
- 对每一个人真诚祝福，而自己"只愿面朝大海"——一种主动的孤独
- 温暖的意象中藏着冷静的距离感

**关键意象**：
- 面朝大海的房子（孤独的栖居）
- 喂马、劈柴（简单纯粹的生活）
- 春暖花开（温暖与希望）
- 幸福的闪电（瞬间的顿悟）
- 给山河取名字（与世界的温柔连接）

**情绪层次**：
1. 对简单幸福的憧憬
2. 对他人真诚的祝福
3. 自己选择的孤独——面朝大海，背对人群

## 设计直觉

### 第一感受

**画面**：清晨，一所面朝大海的小屋，阳光从海平面升起，门前有春天的花朵在微风中摇曳。远处是无垠的蔚蓝，近处是温暖的生活气息。

**色彩**：
- 海天一色的蔚蓝 `#4A90D9`
- 晨曦的暖金色 `#F5D76E`
- 春花的淡粉白 `#FFE4E1`
- 木屋的温暖棕 `#8B7355`
- 草地的新绿 `#90EE90`

**感觉**：温暖中带着淡淡的忧伤，希望中藏着孤独，明亮却不喧闹。

**节奏**：舒缓、呼吸般的节奏，像海浪一样起伏。

## 视觉风格

### 色彩方案

```
主色调：海蓝 #4A90D9 → #87CEEB（天空渐变）
暖色调：晨光金 #F5D76E → #FFF8DC
点缀色：花朵粉 #FFE4E1、草地绿 #90EE90
基础色：沙滩米白 #FFF5EE（背景）
文字色：深海蓝 #2C3E50
```

### 字体选择

- **标题**：手写风格字体（诗歌的个人性）
- **正文**：思源宋体，1.9倍行高（诗歌需要呼吸感）
- **祝福语**：可用更柔和的字体

### 视觉元素

- **海平面**：贯穿始终的视觉锚点
- **小屋剪影**：温暖的家的象征
- **花朵**：点缀的春天气息
- **海浪**：缓慢、有节奏的动态
- **阳光光晕**：温暖的氛围渲染

## 交互设计

### 1. 海浪呼吸

页面背景的海浪以缓慢的节奏起伏：
- 模拟真实海浪的呼吸感
- 6-8秒一个周期
- 配合阅读的自然节奏

### 2. 日出渐变

随着滚动，画面从清晨逐渐变亮：
- 开头：晨曦初露，天边微亮
- 中段：阳光洒满海面
- 结尾：明媚的春日

### 3. 花朵绽放

滚动到"春暖花开"时：
- 画面边缘有花朵渐渐绽放的动画
- 轻微、不喧宾夺主

### 4. 祝福飘散

最后三段祝福语：
- 文字如蒲公英般轻轻飘散
- 或如瓶中信漂向远方
- 传达"告诉每一个人"的意境

## 沉浸式设计

### 核心隐喻：海边的清晨

将阅读体验设计成**站在海边小屋前迎接日出**：

**Hero 区域**（面朝大海）：
- 海天相接的广阔画面
- 一所小屋的剪影
- 标题如同诗人的心声浮现
- 海浪声（可选音效）

**第一节**（从明天起，做一个幸福的人）：
- 小屋逐渐清晰
- 喂马、劈柴的简单意象
- 温暖的晨光开始出现

**第二节**（告诉他们我的幸福）：
- 阳光更加明亮
- "幸福的闪电"——可以有一个光芒闪烁的效果
- 信件/通信的视觉暗示

**第三节**（给每一条河取名字）：
- 画面最为明亮温暖
- 祝福语一句句浮现
- 花朵在角落绽放

**结尾**（我只愿面朝大海）：
- 回到海与天
- 一个人的背影面向大海
- 温暖中带着孤独的宁静

### 双重性的表达

设计要传达这首诗的双重性：
- **表面**：温暖、希望、祝福
- **底层**：孤独、疏离、告别

可以通过：
- 明亮的色调中保留一丝冷色
- 人物始终是背影，面向大海而非观者
- 祝福是"给你们的"，自己选择了孤独

## 响应式考虑

### 桌面端

- 宽幅的海景画面，有足够的留白
- 小屋可以在一侧，海占据大部分画面
- 诗句可以错落排布，如手写信件

### 移动端

- 垂直的海天构图
- 小屋在下方，海天在上方
- 诗句居中排列
- 海浪动画简化以保证性能

## 技术实现

### 组件结构

```
app/components/facing-the-sea/
├── Hero.tsx           # 封面：海边小屋的晨曦
├── ContentSection.tsx # 正文容器：海天背景
├── SeaWaves.tsx       # 海浪动画：CSS/SVG实现
├── Sunrise.tsx        # 日出渐变：随滚动变化
├── Flowers.tsx        # 花朵装饰：绽放动画
└── BlessingText.tsx   # 祝福文字：飘散效果
```

### 关键技术

1. **海浪动画**：CSS animation + SVG path，或 Framer Motion
2. **日出渐变**：useScroll + useTransform 控制背景色
3. **花朵绽放**：SVG 动画或 Lottie
4. **文字飘散**：Framer Motion 的 variants 和 stagger

### 性能优化

- 海浪使用 CSS 动画而非 JS，减少主线程压力
- 花朵等装饰元素懒加载
- 移动端减少同时动画的元素数量
- 使用 will-change 提示浏览器优化

## Hero 组件规格

### inHome={true} 模式（首页卡片）

- 高度固定 `h-[400px]`
- 展示：简化的海边小屋场景
- 海平面、小屋剪影、温暖的天空
- 标题"面朝大海，春暖花开"
- 整体明亮温暖的色调

### inHome={false} 模式（文章详情页）

- 全屏高度 `min-h-screen`
- 完整的海边晨曦场景
- 海浪动画、日出效果
- 可能包含海子的名字和日期

## 特别说明

这首诗有其特殊的历史背景。设计应该：
- 尊重诗歌本身的情感
- 传达温暖但不过度欢快
- 保留那份"面朝大海"的孤独感
- 让读者在温暖中感受到一丝淡淡的忧伤
