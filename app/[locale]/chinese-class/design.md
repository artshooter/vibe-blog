# 「语文课」网页复盘 - 设计文档

## 内容理解

这是一篇项目复盘文章，作者分享了在「语文课」网页项目中使用 AI 文生图的技巧和经验。核心内容包括：

1. **文生图在游戏素材中的应用** - 展示了文生图在创意内容制作中的潜力
2. **实践智慧** - 分享了连续生成、对话管理等技巧
3. **工具对比** - Nana Banana 与 MidJourney 的优缺点分析

## 设计直觉

**第一感受**：这篇文章充满了"实验室感" - 作者在探索、对比、迭代不同的方案。整个阅读过程应该给人一种"跟随创意者的思路"的感觉。

**关键画面**：
- AI 生成的图片在不断迭代改进
- 两个不同工具的对话框，分别展示各自的特点
- 从探索中获得的洞察
- 创意工作室的工作流程

**情绪基调**：好奇、实践、对比、收获

## 视觉风格

### 色彩方案
- **主色**：紫色 (`#8b5cf6`) - 代表 AI 创意、神秘感、高科技
- **辅助色**：蓝色 (`#3b82f6`) - 代表逻辑、清晰、专业
- **强调色**：金色 (`#f59e0b`) 或 琥珀色 - 用于重点提示和闪闪发光的洞察
- **背景**：浅紫色渐变背景或深色背景配紫蓝色点缀

### 字体
- **标题**：使用更现代、几何感强的字体（保持与博客统一）
- **正文**：清晰易读，保持可读性
- **强调**：对比鲜明的强调方式

### 视觉元素
- **装饰线条**：使用几何线条、网格效果，体现"技术"和"创意"的融合
- **微交互**：图片、图表的缓入缓出效果
- **卡片设计**：不同的技巧和对比点用卡片分离，清晰展示

- **展开**：显示详细的解决方案和原理
- **视觉反馈**：卡片展开时有平滑的动画

### 2. 工具对比组件 - 双列对比

- **指标对比**：用可视化的方式展示控制力、审美等维度
  - 控制力（逻辑/准确度）
  - 审美/视觉效果
  - 易用性

### 3. 链接高亮

## 沉浸式设计

**核心隐喻**：创意工作室的实验流程

**布局思路**：
1. **开场**：展示项目概览和链接（邀请进入项目本身）
2. **第一部分**：文生图的应用场景 - 用卡片或网格展示三种可能的游戏类型
3. **第二部分**：技巧分享 - 可展开的卡片形式，像打开一个个秘密盒子
4. **第三部分**：工具对比 - 双列对比，像是一个决策树或评估表
5. **收尾**：回到项目链接，鼓励用户尝试

**视觉流向**：从上到下的纵向流，但在对比部分可以用左右的平衡感。

## 响应式考虑

- **桌面版**：
  - 工具对比采用双列并排
  - 卡片可以有悬停效果
  - 更大的间距和留白

- **移动版**：
  - 工具对比采用上下堆叠（Nana Banana 在上，MidJourney 在下）
  - 卡片简化，展开需要点击而不是悬停
  - 字体大小和间距做出调整，保证易读

## 技术实现建议

1. **展开式卡片**
   - 使用 `<details>` 元素或自定义 React 组件
   - 用 CSS transitions 实现平滑展开动画

2. **对比组件**
   - 在桌面使用 CSS Grid 或 Flexbox 的双列布局
   - 在移动端使用媒体查询切换为单列
   - 考虑用 SVG 或背景渐变创建视觉分割

3. **颜色和渐变**
   - 利用 Tailwind 的渐变功能（`bg-gradient-to-r`）
   - 紫蓝色的渐变背景

4. **微交互**
   - 卡片展开：`max-height` 从 0 到 `auto`
   - 链接按钮：悬停时有阴影或光晕效果

## 特色亮点

1. **金色闪光点**：在关键洞察处（如"解决方案是"）加入细微的金色高亮或动画
2. **实验室感**：用微妙的网格或实验室线条作为背景装饰
3. **清单与小图标**：为列表项添加小图标，增加视觉趣味
4. **引用框**：作者的重要发现用特殊的引用框突出
