# 故乡 - 设计文档

## 内容理解

这是鲁迅的短篇小说代表作。文章以"我"回故乡的见闻为线索，描述了深冬故乡的破败景象和农民闰土的巨大变化。

**核心冲突**：
*   **记忆 vs 现实**：记忆中神异的图画（深蓝天空、金黄圆月、碧绿西瓜） vs 现实中萧索的荒村（苍黄天空、断茎枯草）。
*   **少年 vs 中年**：昔日的小英雄项带银圈、手捏钢叉 vs 今日的木偶人脸色灰黄、松树皮般的手。
*   **纯真 vs 隔膜**：儿时互称"迅哥儿"、"闰土" vs 成年后的"老爷"。

**核心情绪**：
1.  **苍凉 (Desolation)**：主基调，现实的故乡是冷酷、破败的。
2.  **幻灭 (Disillusionment)**：美好记忆的破碎。
3.  **隔绝 (Isolation)**："厚障壁"带来的窒息感。
4.  **微茫的希望 (Faint Hope)**：结尾处关于"路"的哲思。

---

## 设计理念：残梦与荒村

设计不应是温馨的怀旧，而应是**残酷的对比**。试图在灰暗的现实背景上，撕开几道鲜亮的回忆裂口。

### 1. 这种对比如何体现？

*   **色调对立**：
    *   **现实层**（主色调）：低饱和度、压抑。使用 **苍黄**、**灰白**、**深褐**。
    *   **记忆层**（点缀色）：高饱和度、梦幻。使用 **深蓝**（夜空）、**金黄**（圆月）、**碧绿**（瓜田）。
*   **质感对立**：
    *   现实部分使用由于风化而粗糙的质感（如枯草、旧墙）。
    *   记忆部分使用清透、明亮、无杂质的质感。

## 视觉风格

### 色彩系统

```css
/* 现实层 - 萧索基调 */
--bg-main: #f0f0f4;        /* 阴晦的天色/冻云灰 */
--bg-card: #e8e8e6;        /* 旧屋墙壁灰 */
--text-primary: #2d3336;   /* 墨黑 */
--text-secondary: #6e757c; /* 尘土灰 */
--border-divider: #d1d5db; /* 枯草白 */

/* 记忆层 - 梦境亮色 */
--memory-blue: #1b3a57;    /* 深蓝天 */
--memory-gold: #f2c04d;    /* 金黄月 */
--memory-green: #2d5a3f;   /* 碧绿瓜田 */

/* 情感层 */
--accent-barrier: #8c3f3f; /* 隔膜（暗红/锈色） */
```

### 字体排版

*   **正文**：使用衬线体（Serif），增加文学沉淀感。行间距略大，营造疏离感。
*   **对话**：
    *   **"我"**：正常衬线体。
    *   **少年闰土**：活泼的、略带圆角的字体（或手写体感）。
    *   **中年闰土**：其对话框带有"裂痕"或"风化"效果，字体压抑。

## 核心组件设计

### 1. Hero：神异的图画 (The Dream)

*   **视觉**：**全屏沉浸式**。
    *   背景是纯净的**深蓝**。
    *   正中央悬挂一轮**金黄圆月**。
    *   底部是剪影般的**碧绿沙地**。
    *   **交互**：鼠标移动时，少年刺猹的剪影在瓜田中隐现，仿佛捉摸不定的记忆。
*   **标题**：巨大的竖排「故乡」，字体边缘带有轻微的风化侵蚀效果。

### 2. 现实切入：萧索荒村 (The Reality)

*   **转场**：从Hero滑落，画面瞬间转为**苍黄阴晦**。
*   **背景特效**：页面两侧偶有枯草断茎随滚动轻微摇曳，模拟"冷风吹进船舱"的寒意。

### 3. 组件：「厚障壁」 (The Barrier)

*   **位置**：文章中闰土喊出"老爷"的那一刻。
*   **交互行为**：
    *   用户滚动到此句时，屏幕中央出现一道**物理裂痕**或**半透明的高墙**。
    *   **左侧**：浮现少年闰土的幻象（色彩鲜艳，项带银圈）。
    *   **右侧**：站立中年闰土的立绘（灰黄单薄，只有眼圈红肿）。
    *   **文字**："我们之间已经隔了一层可悲的厚障壁了" —— 这行字像钉子一样钉在裂痕上。

### 4. 结尾：「路」 (The Hope)

*   **视觉**：背景逐渐暗淡，回到深蓝色的朦胧中。
*   **文字**："其实地上本没有路，走的人多了，也便成了路。"
*   **交互**：文字呈现微弱的**呼吸光效**，仿佛黑夜中的一点烛火。随着用户停留，光晕稍微扩散，象征希望的蔓延。

---

## 响应式策略

*   **移动端**：
    *   Hero 保持视觉冲击力，金黄圆月居中。
    *   「厚障壁」组件改为上下布局，"墙"横亘在两个闰土之间。
*   **桌面端**：
    *   利用宽屏强化"疏离感"，文字两侧留白更多。
    *   「厚障壁」组件左右对峙。

---

## 总结

新的设计不再追求"优美"，而是追求**震撼与真实**。通过色彩和交互的强烈反差，让读者直观感受到鲁迅笔下那个"悲凉"的故乡和"麻木"的国人，最后在微弱的希望中结束。
