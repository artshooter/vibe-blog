# 《雨巷》设计文档

## 内容理解
戴望舒的成名作，是中国现代诗歌的经典之作。全诗通过"雨巷"、"油纸伞"、"丁香"、"姑娘"等意象，营造出一种凄清、朦胧、哀怨的氛围。诗人表达了一种在理想与现实之间挣扎、彷徨的心境。诗歌节奏舒缓，重叠和反复的运用增强了音乐美感。

## 设计直觉
- **画面**：青石板小巷、细雨朦胧、油纸伞的剪影、颓圮的篱墙。
- **色彩**：青灰色（雨巷）、淡紫色（丁香）、忧郁蓝（雨水）、灰白色（薄雾）。
- **感受**：凄清(desolate)、朦胧(misty)、惆怅(melancholy)、彷徨(hesitant)。

## 视觉风格
- **色彩(Colors)**:
  - 背景：深灰色石板纹理，覆盖一层半透明的青蓝色层 (#2F4F4F -> #708090)。
  - 主色：丁香紫 (#D8BFD8)，用于点缀和关键意象。
  - 文字：淡灰白色 (#E0E0E0)，在深色背景上营造出一种轻盈感。
- **字体(Fonts)**: 正文使用极细的非衬线体或具有文艺感的宋体，保持诗歌的呼吸感。
- **视觉元素**: 细雨飘落的动态粒子、石板路上的倒影、丁香花的剪影、模糊的景深效果。

## 交互设计
1. **雨声环境音**: 自动播放轻微的下雨声，声音随滚动深度产生细微的变化（如从巷口到巷深处的声场变化）。
2. **"丁香姑娘"的消散**: 随着用户的滚动，背景中的丁香姑娘剪影会逐渐变得 transparent(透明) 甚至消散，体现诗中"消了她的颜色，散了她的芬芳"。
3. **文字层叠感**: 利用 CSS 视差效果，文字层与背景的"篱墙"、"丁香"产生不同步的移动，增加巷道的深度感。
4. **油纸伞动效**: 页面顶部的 Hero 组件中，油纸伞会随着鼠标移动产生轻微的 tilt(倾斜) 效果。

## 沉浸式设计（隐喻）
**隐喻：一段无法走出的悠长巷道**
页面被设计成一条无尽向下的石板路。滚动的过程就像是在雨巷中"彳亍(chichu)"。两侧的文字布局不规则，模拟出巷道转弯或不平整的感觉。

## 响应式考虑
- **桌面端**: 增加两侧的"颓圮篱墙"装饰，增强空间感。
- **移动端**: 侧重于雨滴粒子在屏幕上的交互，文字居中排列，通过背景色的明暗变化来传达情绪。

## 技术实现
- **Framer Motion**: 控制文字和姑娘剪影的消散动画。
- **Three.js / CSS Particles**: 实现逼真的雨丝下落和地面溅起的水花效果。
- **Intersection Observer**: 精确触发每一节诗句对应的背景意象变化。
