# 孔乙己 - 设计文档

## 1. 内容理解

### 主题
鲁迅经典短篇小说。孔乙己是"站着喝酒而穿长衫的唯一的人"——封建科举制度下的边缘人，被众人嘲笑取乐，最终悄无声息地消失。

### 情绪
- 表面：酒馆的"快活"气氛
- 内里：深沉的悲凉与讽刺
- 核心张力："店内外充满了快活的空气"——以他人痛苦为乐

---

## 2. 视觉风格

### 整体定位
**丰子恺式白描 + 国画点彩**

### 线条
- 黑色毛笔笔触
- 寥寥数笔，简洁传神
- 人物轮廓、环境勾勒均为黑白

### 色彩
- **底色**：米白/宣纸色 `#F8F4E8`
- **线条**：墨黑 `#1A1A1A`
- **点彩**（仅重点部位）：
  - 茴香豆：赭黄 `#C4A35A`
  - 长衫：青灰/靛蓝 `#4A5568`（破旧褪色感）
  - 脸色：青白 → 涨红 `#B85C5C` → 黑瘦
  - 黄酒：琥珀 `#D4A84B`
  - 铜钱：古铜 `#8B7355`

### 留白
大面积留白，呼吸感强，符合国画审美

---

## 3. 阅读体验设计

### 节奏
```
正常文字阅读
    ↓ 滚动
场景插画（占满视口）
    ↓ 滚动
动画播放 + 对话出现
    ↓ 滚动
回到正常文字
```

### 对话呈现
- **无对话框**
- 毛笔字体的对话文字
- 一条细线连接文字与说话的人物（丰子恺风格）

---

## 4. 四个核心场景

### 场景一：初登场
**位置**：孔乙己第一次出现后

**画面**：
- 咸亨酒店曲尺形柜台（简笔勾勒）
- 孔乙己站立，瘦高身材，破旧长衫，花白胡子
- 一手端碗，一手排钱

**动画**：
- 滚动触发：手掌摊开，九文大钱一枚枚排出
- 铜钱有古铜色点彩

**对话**：
- "温两碗酒，要一碟茴香豆。"

---

### 场景二：教"回"字
**位置**：孔乙己教小伙计写字后

**画面**：
- 柜台一角
- 孔乙己侧身，两根手指敲着柜台
- 小伙计背对（或侧脸，表情不耐烦）

**动画**：
- 滚动触发：指甲蘸酒，在柜台上写出"回"字
- 字迹用淡墨渲染，慢慢晕开消失

**对话**：
- "回字有四样写法，你知道么？"

**彩蛋**：可点击查看四种写法：回、囘、囬、廻

---

### 场景三：分茴香豆
**位置**：孔乙己给孩子们分豆后

**画面**：
- 孔乙己弯腰，五指罩住碟子
- 三四个孩子围着他，眼睛都望着碟子
- 碟中茴香豆（赭黄色点彩）

**动画**：
- 滚动触发：孔乙己直起身，看碟子，摇头
- 豆子从碟中一颗颗减少

**对话**：
- "不多了，我已经不多了。"
- "不多不多！多乎哉？不多也。"

---

### 场景四：最后一次
**位置**：孔乙己最后出现的段落

**画面**：
- 门槛下，孔乙己坐着
- 脸黑而瘦，破夹袄
- 盘着两腿，下面垫蒲包
- 双手沾满泥

**动画**：
- 滚动触发：从破衣袋摸出四文大钱
- 然后"坐着用这手慢慢走去"——双手撑地，身体缓缓移出画面

**对话**：
- "温一碗酒。"（声音极低）
- "跌断，跌，跌……"

**色彩变化**：
- 整体色调变冷、变灰
- 脸色从青白变为黑瘦（灰色渲染）

## 5. 技术实现

### 组件结构
```
app/components/kong-yiji/
├── index.tsx              # 导出
├── Content.tsx            # 文章主体（组织文字和场景）
├── scenes/
│   ├── Scene1Debut.tsx    # 初登场
│   ├── Scene2HuiChar.tsx  # 教"回"字
│   ├── Scene3Beans.tsx    # 分茴香豆
│   └── Scene4Final.tsx    # 最后一次
├── HuiCharacter.tsx       # "回"字四写法（可选交互）
└── DialogueLine.tsx       # 对话组件（线连接人物）
```

### 动画方案
- 使用 `framer-motion` 的滚动动画
- SVG 路径动画实现毛笔线条效果
- `useScroll` + `useTransform` 控制场景进度

### 插画资源
- SVG 格式，支持动画
- 可考虑：
  - 手绘后矢量化
  - 纯 SVG 路径绘制
  - AI 生成 + 手动调整

### 响应式
- 桌面端：场景插画占满视口，左右留白
- 移动端：场景插画全宽，上下留白
- 文字部分保持舒适阅读宽度
