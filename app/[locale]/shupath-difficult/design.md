# 蜀道难 - 设计文档

## 内容理解

李白的乐府名篇，以雄奇豪放的笔触描绘蜀道的艰险，融神话、历史、自然于一体，气势磅礴，惊心动魄。

**结构层次**：
1. **开篇**：感叹蜀道之高，追溯开国传说
2. **中段**：描绘旅途艰险，悲鸟子规，使人凋朱颜
3. **结尾**：剑阁雄关的政治隐喻，劝人早归

**三叹"蜀道难"**：
- 第一叹：难于上青天（高度之极）
- 第二叹：使人听此凋朱颜（险峻之惧）
- 第三叹：侧身西望长咨嗟（前路之忧）

**关键意象**：
- **高**：六龙回日、扪参历井、连峰去天不盈尺
- **险**：黄鹤不得过、猿猱愁攀援、枯松倒挂
- **急**：冲波逆折、飞湍瀑流、万壑雷鸣
- **危**：剑阁峥嵘、猛虎长蛇、杀人如麻

## 设计直觉

### 第一感受

**画面**：仰望，无尽的山峰刺破云霄；俯视，深渊万丈，激流轰鸣。栈道如细线悬于绝壁，行人如蚁。云雾翻涌，遮天蔽日。

**色彩**：
- 山石的青灰 `#4A5568`
- 深渊的墨黑 `#1A202C`
- 云雾的苍白 `#E2E8F0`
- 古松的苍绿 `#2D3B2D`
- 瀑布的冷白 `#F7FAFC`
- 点缀的朱红 `#C53030`（警示、危险）

**感觉**：震撼、敬畏、压迫感、窒息般的险峻

**节奏**：跌宕起伏，如攀登般时而陡峭时而喘息

## 视觉风格

### 色彩方案

```
主色调：山岩青灰 #4A5568 → #2D3748（层峦渐变）
深色调：深渊墨黑 #1A202C（底部、阴影）
亮色调：云雾白 #E2E8F0 → #F7FAFC
点缀色：朱红 #C53030（危险警示）、苍松绿 #2D3B2D
```

### 字体选择

- **标题**：粗犷的书法体（气势磅礴）
- **"噫吁嚱"**：特大号，可用草书风格
- **正文**：楷体或宋体，适当加粗
- **重复句**：每次出现可递进放大

### 视觉元素

- **层叠山峦**：用剪影或渐变表现远近
- **云雾翻涌**：半透明的动态云层
- **栈道悬空**：细线般的路径
- **瀑布飞流**：动态的水流效果
- **枯松倒挂**：悬崖上的孤松剪影

## 交互设计

### 1. 垂直攀登

整个页面模拟攀登蜀道的体验：
- 滚动方向即攀登方向
- 越往下滚动，视觉上越"高"
- 背景山峦产生视差效果

### 2. 云雾穿行

滚动时云雾层飘动：
- 模拟穿越云层的感觉
- 有时遮挡视线，增强神秘感
- 使用多层半透明元素

### 3. 三叹递进

每次出现"蜀道之难，难于上青天"：
- 字号递进变大
- 视觉效果递进增强
- 第一次：远山回响
- 第二次：悬崖震动
- 第三次：全屏震撼

### 4. 声景效果（可选）

- 风声呼啸
- 瀑布轰鸣
- 猿猴啼叫
- 子规夜啼

### 5. 险峻可视化

特定意象的增强展示：
- "黄鹤之飞尚不得过"：一只鹤的剪影试图飞过却折返
- "枯松倒挂倚绝壁"：悬崖上倒挂的松树
- "飞湍瀑流"：动态的瀑布
- "一夫当关"：雄关剪影

## 沉浸式设计

### 核心隐喻：攀登蜀道

将阅读体验设计成一次**惊心动魄的蜀道之旅**：

**Hero 区域**（仰望蜀道）：
- 从低处仰望高耸入云的山峰
- "噫吁嚱，危乎高哉！"震撼登场
- 云雾缭绕，遮挡山顶
- 让人感受到即将踏上的旅途之险

**第一段**（历史与传说）：
- 蚕丛鱼凫的远古剪影
- 四万八千岁的时间感
- 天梯石栈逐渐显现
- 色调偏古朴、神秘

**第二段**（旅途艰险）：
- 画面变得更加险峻
- 悲鸟号古木、子规啼夜月
- 连峰去天、枯松倒挂
- 瀑布飞流、万壑雷鸣
- 视觉和听觉的双重冲击

**第三段**（剑阁雄关）：
- 剑阁的雄伟关隘
- 猛虎长蛇的危险暗示
- 色调转为警示的暗红
- "不如早还家"的规劝

**结尾**（回望长叹）：
- 侧身西望的身影
- 蜀道在身后蜿蜒
- 长叹一声，余音回响

### 纵深感的营造

- **多层视差**：近景栈道、中景山峰、远景云天
- **透视夸张**：强调高度和深度
- **尺度对比**：人如蚂蚁，山如天柱

## 响应式考虑

### 桌面端

- 宽幅画面，展现山峦的横向延伸
- 视差滚动效果完整呈现
- 云雾动画更加细腻
- 可在侧边显示攀登进度

### 移动端

- 纵向构图，强调高度感
- 简化云雾层数，保证性能
- 手指滑动即"攀登"
- 三叹效果用震动反馈增强（如果支持）

## 技术实现

### 组件结构

```
app/components/shupath-difficult/
├── Hero.tsx              # 封面：仰望蜀山
├── ContentSection.tsx    # 正文容器
├── MountainLayers.tsx    # 层叠山峦：视差效果
├── CloudMist.tsx         # 云雾动画
├── Waterfall.tsx         # 瀑布效果
├── CliffPath.tsx         # 栈道/剑阁
├── ThreeSighs.tsx        # 三叹递进效果
└── SilhouetteFigure.tsx  # 行人/鹤/松等剪影
```

### 关键技术

1. **视差滚动**：多层背景不同速度移动
2. **云雾效果**：CSS filter blur + 动态 transform
3. **瀑布动画**：SVG 或 CSS animation
4. **三叹效果**：Framer Motion 的 scale + shake
5. **剪影图形**：SVG 矢量图，便于缩放

### 性能优化

- 山峦用 CSS 渐变而非图片
- 云雾层控制在 3-4 层
- 瀑布用 CSS 动画而非视频
- 移动端关闭部分视差效果
- 使用 Intersection Observer 按需加载动画

## Hero 组件规格

### inHome={true} 模式（首页卡片）

- 高度固定 `h-[400px]`
- 展示：层叠山峦的剪影，云雾缭绕
- 标题"蜀道难"以粗犷书法风格呈现
- 副标题或首句"噫吁嚱，危乎高哉！"
- 色调以青灰为主，给人险峻感

### inHome={false} 模式（文章详情页）

- 全屏高度 `min-h-screen`
- 从低处仰望的完整山景
- 云雾动态飘动
- "噫吁嚱"以震撼方式登场
- 引导用户向下滚动"攀登"

## 特别说明

### 气势的表达

李白此诗最突出的是**气势**：
- 开篇的"噫吁嚱"是前所未有的感叹
- 夸张的数字（四万八千岁、百步九折）
- 密集的险峻意象堆叠

设计要传达这种**压倒性的力量感**：
- 字体要有力度
- 色调要有重量
- 空间要有压迫感
- 动效要有冲击力

### 古典与现代的平衡

- 保留水墨山水的意境
- 用现代技术增强沉浸感
- 不要过于写实，保留想象空间
- 剪影和留白比细节更有力
