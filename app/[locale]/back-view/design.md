# 《背影》设计文档

## 内容理解

朱自清的《背影》是一篇经典散文，讲述了父子之间深沉而含蓄的爱。文章核心场景是父亲在火车站为"我"买橘子时攀爬月台的背影——那个肥胖、穿着黑布马褂青布棉袍、蹒跚而努力的身影，成为了父爱最动人的象征。

**核心主题**：
- 父爱的深沉与无言
- 时光流逝与世事变迁
- 成长后对父亲的理解与愧疚
- 离别与思念

**关键意象**：
- 背影：父爱的象征，贯穿全文
- 橘子：朱红色，父爱的具体呈现
- 火车站月台：离别的场所
- 黑布马褂、青布棉袍：时代与朴素
- 泪水：情感的流露

## 设计直觉

**第一感受**：
阅读这篇文章，脑海中浮现的是一个略显模糊的、渐行渐远的背影。色调是灰蓝色的冬日、暗沉的黑色衣物、以及那一抹温暖的朱红橘子色。整体氛围是克制的、沉默的，却暗涌着深厚的情感。

**画面**：
- 老旧火车站的月台
- 冬日的清冷光线
- 一个蹒跚的身影在人群中渐渐远去
- 泪眼朦胧中的模糊轮廓

**情绪**：沉静、温暖、略带忧伤、怀旧

## 视觉风格

### 色彩方案

采用低饱和度的怀旧色调，突出文章的时代感和情感克制：

```
主色调：
- 深灰蓝 #2C3E50 - 冬日的冷清
- 暖灰 #7F8C8D - 老旧的质感
- 米白 #F5F5DC - 纸张的温度

点缀色：
- 朱红 #D2691E - 橘子，温暖的父爱
- 墨黑 #1a1a1a - 父亲的衣着

暗色模式：
- 背景 #0d1117 - 深夜的回忆
- 文字 #c9d1d9 - 柔和的阅读
- 点缀 #d97706 - 橘子的温暖
```

### 字体选择

- **正文**：衬线字体，如 `Noto Serif SC`，传达经典文学的韵味
- **标题**：略粗的衬线体，沉稳有力

### 视觉元素

- **背景**：模糊的、渐隐的剪影效果，象征记忆的模糊
- **纹理**：轻微的纸张纹理或老照片质感
- **装饰**：可用淡淡的水墨笔触作为分隔线

## 交互设计

### 核心交互：背影的渐隐

**背影剪影动画**：
- 页面加载时，一个父亲的剪影从清晰到模糊
- 随着用户向下滚动，背影逐渐融入背景
- 象征"混入来来往往的人里，再找不着了"

### 买橘子场景可视化

**月台攀爬动画**：
- 当读到父亲买橘子的段落时
- 触发一个简单的剪影动画：攀爬、努力、往回走
- 用CSS动画实现，保持简洁

### 泪水效果

**情感高潮提示**：
- 在"我的泪很快地流下来了"等关键句子
- 轻微的模糊效果，模拟泪眼朦胧
- 或页面边缘出现淡淡的水波纹

### 时间线导航

**文章时间轴**：
- 侧边显示时间线：祖母去世 → 奔丧 → 火车站送别 → 多年后读信
- 帮助读者理解故事的时间脉络

## 沉浸式设计

### 隐喻：渐行渐远的背影

整个页面采用"背影渐隐"的隐喻：

**视觉层次**：
1. **开篇**：清晰的父亲剪影，近在眼前
2. **中段**：随着阅读，剪影开始后退、缩小
3. **买橘子高潮**：背影最清晰的瞬间
4. **结尾**：背影融入模糊的背景，"再找不着了"

**滚动效果**：
- 使用视差滚动
- 背影元素固定，但随滚动变化透明度和模糊度
- 营造"追忆"的感觉

### 场景氛围

**Hero区域**：
- 老旧火车站月台的氛围
- 灰蓝色调的冬日光线
- 一个渐隐的背影剪影

## 响应式考虑

### 桌面端
- 左侧可放时间线导航
- 正文居中，宽度适中（约 700px）
- 背影剪影可作为固定背景元素

### 移动端
- 时间线改为顶部水平滚动或隐藏
- 全宽阅读，更沉浸的体验
- 动画效果简化，保证性能
- 触摸滑动触发背影变化

## 技术实现

### 组件结构

```
back-view/
├── page.tsx          # 页面主组件
├── content.md        # 文章内容
├── design.md         # 设计文档
└── components/
    ├── Hero.tsx              # 首页展示组件
    ├── BackViewHero.tsx      # 文章Hero，带背影剪影
    ├── ArticleContent.tsx    # 正文内容
    ├── SilhouetteAnimation.tsx # 背影剪影动画
    ├── Timeline.tsx          # 时间线导航
    └── PlatformScene.tsx     # 月台场景可视化
```

### 关键技术

1. **背影剪影**：CSS clip-path + SVG 路径，创建父亲轮廓
2. **渐隐效果**：CSS filter: blur() + opacity，响应滚动
3. **视差滚动**：Framer Motion 或 CSS scroll-linked animations
4. **泪水模糊**：backdrop-filter 或 canvas 效果

### 动画性能

- 使用 CSS transform 和 opacity 确保 GPU 加速
- 避免复杂的 JS 动画计算
- 移动端简化或禁用部分效果
